# Пример файла README.md
 
## Введение
 
Всем здравствуйте! Это пример файла README.md, здесь будет немного возможностей разметки **Markdown**:smiling_imp:.
 
### Основные элементы разметки
 
#### Заголовки
 
Благодаря использованию заголовков разных уровней, получается удобно структурировать текст (их всего 6: от # и до ######).
 
#### Курсив и жирный шрифт
 
Использование жирного и наклонного шрифта, помогает выделить важные моменты:
 
- *Курсив* используется для акцентирования внимания на определенных моментах;
- **Жирный шрифт** используется для выделения важных слов или фраз в текстах или предложениях;
- ***Жирный курсив*** используется для выделения самых важных слов или фраз;
- ~Зачеркнутый~ он прикольный, пусть тоже будет.
 
#### Цитаты и изображения
 
Цитаты подчеркивают важные мысли или высказывания:
 
> Можно заехать неизвестно куда… лишь бы знать, откуда.

Кстати это цитата из мультфильма `Тачки`.

#### Ссылки
 
Использую ссылки, чтобы предоставить дополнительную информацию или направить читателя на какой-то ресурс: [alliancek](https://github.com/alliancek).

#### Кодовые блоки и подсветка синтаксиса
 
Кодовые блоки для вставки фрагментов кода в Markdown поддерживают подсветку синтаксиса для различных языков программирования.
 
```python
def hello_world():
    print("Hello, world!")
```
***
```javascript
function helloWorld() {
  console.log("Hello, world!");
}
```

#### Таблицы
 
Cоздание таблицы представляет данные в структурированном виде.
 
| Шрифт         | Обозначение | Использование     |
|-|:-:|-:|
| курсив       | *        |Для длинных текстов    |
| жирный       | **       |Для акцентов в тексте  |
| жирный курсив| ***      |Для творческих проектов|
 

#### Списки
 
С помощью списков можно сразу перечислить несколько пунктов:

1. Пункт 1
2. Пункт 2
3. Пункт 3
 
Или вложенные списки:
 
- Пункт 1
  - Подпункт 1.1
  - Подпункт 1.2
- Пункт 2
  - Подпункт 2.1
  - Подпункт 2.2
- Пункт 3

Создание списка задач с галочками:
 
- [x] Задача 1 (выполнено)
- [ ] Задача 2 (не выполнено)
- [x] Задача 3 (выполнено)
 
## Заключение
